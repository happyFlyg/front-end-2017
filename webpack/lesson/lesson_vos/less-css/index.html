js压缩

  1. cnpm install uglifyjs -g
  2. cnpm install uglifyjs --save-dev
  3. uglifyjs index.js -o index.min.js
  4. 先混淆后压缩 uglifyjs index.js -m -o index.min.js。注意：-m在前面
