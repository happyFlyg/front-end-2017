图片压缩：

  1. cnpm init 初始化(创建json文件)
  2. cnpm install node-smushit -g
  3. smushit img 图片没有压缩，300dpi会很明显
  1. cnpm install base64-css -g
  2. cnpm install base64-css --save-dev
  3. 非全局安装调用命令：node node_modules/base64-css/bin/cli.js -f index.css

注释：

  1. 帮助：smushit -h
  2. tinify 收费的
  3. base-64-css
  4. cli是命令提示行
