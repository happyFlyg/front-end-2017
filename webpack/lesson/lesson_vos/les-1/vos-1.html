1. cnpm init 初始化
  2. cnpm install babel-cli --save-dev
  3. babel的配置文件：新建.babelrc文件
  4. .babelrc文件里面写：{"presets":["es2015"],"plugins":[]}
  5. 安装es5标准：cnpm install --save-dev babel-preset-es2015
  6. babel es6.js -o es5.js
  7. babel es6 -d es5
  8. babel --watch es6 -d es5
  9. ctrl+C 终止批处理
  10. 在package.json scripts 里面设置 "build":"babel es6 -d lib"
