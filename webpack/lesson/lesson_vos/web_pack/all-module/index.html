引入jquery模块

  1. cnpm install jquery -g

css 打包

  1. cnpm install css-loader --save-dev
  2. cnpm install style-loader --save-dev
  3. 				module:{
					loaders:[
					   {
					   	test:/\.css$/,
					   	loader:'style-loader!css-loader'
					   }
					]
				}

less：

  1. 				module:{
					loaders:[
					   {
					   	test:/\.css$/,
					   	loader:'style-loader!css-loader'
					   },
					   {
					   	test:/\.less$/,
					   	loader:'style!css!less'
					   }
					]
				}

css中嵌套另外一个css

  1. 主文件写：@import url('./font.css')

less less-loader：

  1. 同时安装less-loader和less：cnpm install less-loader less --save-dev

  1. 引用时主模块引用就够了，子模块不能引用，不然打包后会非常庞大
