步骤：

  1. cnpm init -y
  2. cnpm install webpack --save-dev
  3. 安装jquery依赖：cnpm install jquery --save

webpack-dev-server：

  1. cnpm install webpack-dev-server -g
  2. cnpm install webpack-dev-server --save-dev
  3. 				module.exports={
					entry: __dirname + "/src/js/index.js",
					output: {
						path: __dirname + "/assets/js",
						filename: "index.js",
						publicPath:"/temp/"
					},
					devServer:{
						contentBase:"./",
						host:"192.168.1.125",
						port:'8080',
						color:true
					}
				}
  4. 最后访问：http://192.168.1.125:8080/
  5. 实时编译刷新：webpack-dev-server --inline或者webpack-dev-server --iFrame
  6. 计算哪个页面的哪个部分改变了： webpack-dev-server --inline --hot

注意:

  1. --save-dev（开发环境的依赖）与--save（生产环境依赖）。输入cnpm install安装两种依赖，输入cnpm install --production安装save依赖
  2. cnpm init -y 加上y后直接生成package.json文件
  3. cnpm install webpack@1 -g 安装1.0版本@1
  4. cnpm install webpack-dev-server@1 -g
  5. 引用时引用缓存里面的temp
  6. 当使用--hot（只刷新改变的部分，常用于spa页面）时，要引入webpack. 
    			var webpack=require('webpack');
  7. webpack 在服务端使用热查，像淘宝大型网站可以用，常用前端框架都有热查功能
