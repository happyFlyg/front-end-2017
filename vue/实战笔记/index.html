1. 组件化模块化的优势是代码的复用性，可维护性，可扩展性

前期准备：

  1. 安装node
  2. cnpm install vue -g 全局安装vue 之后输入vue list 查看
  3. vue init webpack sell 创建一个基于webpack的工程sell
  4. cd sell
  5. cnpm install
  6. cnpm run dev
