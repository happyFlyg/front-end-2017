Vue-loader搭建自己的webpack

笔记

  1. cnpm init -y
  2. cnpm install webpack --save-dev
  3. cnpm install css-loader extract-text-webpack-plugin html-webpack-plugin json-loader less less-loader style-loader uglify-js webpack webpack-dev-server --save-dev

注意：

  1. 在github上fork一下webpack
  2. 安装的方式down到本地来：命令行 输入 vue init happyFlyg/webpack temp：其中temp是项目名称
  3. 假若只有package.json文件则只写cnpm install就可以安装所有的依赖
