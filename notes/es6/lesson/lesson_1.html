1. export命令：外部读取模块内部的变量，必须用export输出内部变量

    1. export var firstName='liao';
    2. export var lastName="li";
    3. export var year="1990";

  2. export 可以合并书写:export{firstName,lastName,year};

    1. var firstName='liao';
    2. var lastName="li";
    3. var year="1990";

    优先考虑这种写法，因为在脚本尾部一眼就能看出输出了哪些变量

  3. export 1 会报错， var m=1; export m;也会报错,这种写法只是输出的数字，没有输出对外的接口

    正确的写法如下：
    1. export var m=1;
    2. var m=1;export{m};
    3. var m=1; export{m as n};

  4. 总结一下export用法：export{a,b,c}，export{fn1,fn2,fn3}，其中a,b,c是变量，fn是函数
